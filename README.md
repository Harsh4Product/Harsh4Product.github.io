# FM Harsh Profile Website

Single-page professional profile website based on the supplied website build document.

## Sections

- Hero with professional positioning, CTA buttons, skills ticker, and metrics strip.
- Philosophy section with five product leadership principles.
- Experience timeline.
- Product surfaces and rapid AI prototype cards.
- Skills, certifications, side projects, and contact CTA.

## Run Locally

From this folder:

```powershell
.\start.ps1
```

Then open:

```text
http://127.0.0.1:4173
```

If Node.js is installed normally, this also works:

```powershell
npm start
```

## Personalization

Most site content is stored in `app.js` as editable arrays and profile fields. Replace the email, LinkedIn URL, real experience, certifications, and metrics when final details are ready.
